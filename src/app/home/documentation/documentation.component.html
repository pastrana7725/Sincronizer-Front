How to use it?

  1. Requeriments

    1. Microsoft 365 account

      • User requires microsoft 365 work or school account. Microsoft personal accounts do not have the necessary permissions for this operation.

      • The user must know the group id to which they want to synchronize.

    2. QDPM server

      • User requires access to a QDPM server. This means having a QDPM account at your fingertips.

      • The user needs to know the IP address in which the QDPM database is running, the port, username and password
